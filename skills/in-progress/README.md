# 开发中

仍在开发中的技能。尚未达到发布标准——可能存在粗糙的边界、破坏性变更和被放弃的实验。在毕业到稳定版之前，这些技能不会包含在插件和顶层 README 中。

- **[review](./review/SKILL.md)** — 从固定点开始沿两条平行轴线审查变更：**规范**（diff 是否遵循仓库的编码规范？）和**规格**（diff 是否忠实地实现了原始 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 将文章塑造成一段节拍之旅，类似分支选择式冒险。选取一个起始节拍，只写那一拍，然后转向下一拍，直到文章自然收尾。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 通过盘问来挖掘碎片——异质化的写作片段——并将它们追加到一份文档中，作为未来文章的原始素材。
- **[writing-shape](./writing-shape/SKILL.md)** — 拿一份 markdown 格式的原始素材，通过对话逐段将其塑造为一篇文章，每一步都要论证格式选择的合理性。
