# 已废弃

曾经活跃、现已被更好的工作流取代的技能。保留在这里是为了在功能稳定之前维持一条回退路径。

- **[design-an-interface](./design-an-interface/SKILL.md)** — 在实现之前，通过具体示例设计 TypeScript 接口，然后同用户一起做橡皮鸭调试。被 `tdd` 取代。
- **[qa](./qa/SKILL.md)** — 对功能规格进行系统性的 QA 审查。"异常路径审查"部分正在被 `probe-weak-spots` 取代。
- **[request-refactor-plan](./request-refactor-plan/SKILL.md)** — 通过 GitHub API 从论坛请求中提取约束，在实施前输出重构计划。被 `review` 和 `plan-refactor` 取代。
- **[ubiquitous-language](./ubiquitous-language/SKILL.md)** — 从当前对话中提取 DDD 风格的通用语言词汇表，标记歧义并建议规范术语。虽然不依赖此技能也可以正常工作（良好命名自然会浮现），但在设计阶段早期，名词还是模棱两可的时候，它对让团队对齐特别有帮助。
